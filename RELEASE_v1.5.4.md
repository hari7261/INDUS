# INDUS v1.5.4 - Banner Animation Polish

INDUS `v1.5.4` is a focused polish release for the Windows GUI experience.

## Highlights

- Removes the top startup wave lines for a cleaner launch screen
- Adds a right-side robot hand-wave animation during startup
- Preserves the tricolor INDUS banner styling
- Keeps UTF-8 console handling for cleaner Unicode rendering in Windows terminals
- Continues the verified command compatibility from the previous production release

## Included Quality Checks

- Windows GUI binary rebuilt from the latest source
- Command smoke suite rerun against the new executable
- Release assets packaged for Windows distribution

## Upgrade Notes

- Existing command fixes from `v1.5.3` remain included
- This release mainly improves the startup presentation and packaging consistency
