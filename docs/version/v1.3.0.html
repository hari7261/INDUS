Released February 26, 2026

v1.3.0 was the pre-registry INDUS shell.

The 1.3.0 line exposed a smaller built-in set around init, run, http, and version, then fell back to host shell commands for anything else.

Legacy characteristics

  • Single-page documentation site
  • No central registry JSON
  • Windows-focused entrypoint code
  • Mixed native and shell execution model

Built-in commands

  • init project scaffold
  • run simulated workload runner
  • http HTTP GET/POST/PUT/DELETE
  • version release output

Migration to 1.4.0

  • init becomes ind proj init
  • run maps to ind dev bench
  • http becomes ind net fetch
  • Shell fallback is removed from the native path
Legacy snapshot retained for migration context.
